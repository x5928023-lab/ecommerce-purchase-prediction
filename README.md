# E-commerce Purchase Prediction

## Overview
This project predicts customer purchase intention using session-level e-commerce data.

The goal is to identify which users are likely to make a purchase, enabling better targeting, personalization, and revenue optimization.

---

## Problem Statement
E-commerce platforms need to understand user behavior to improve conversion rates.

This project builds machine learning models to classify whether a session will result in a purchase.

---

## Key Features

- Data preprocessing and cleaning  
- Feature engineering from session data  
- Handling class imbalance  
- Training and evaluating multiple ML models  
- Comparing performance across models  

---

## Models Used

- Logistic Regression  
- Random Forest  
- Support Vector Machine (SVM)  
- XGBoost  

---

## Evaluation Metrics

- Accuracy  
- Precision  
- Recall  
- F1-score
